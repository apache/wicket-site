Apache Wicket 7.16.0 released

08 Jan 2020

The Apache Wicket PMC is proud to announce Apache Wicket 7.16.0!

Apache Wicket is an open source Java component oriented web application framework that powers thousands of web applications and web sites for governments, stores, universities, cities, banks, email providers, and more. You can find more about Apache Wicket at https://wicket.apache.org

This release marks another minor release of Wicket 7. We use semantic versioning for the development of Wicket, and as such no API breaks are present breaks are present in this release compared to 7.0.0.

New and noteworthy ------------------ Using this release ------------------ With Apache Maven update your dependency to (and don't forget to update any other dependencies on Wicket projects to the same version): Or download and build the distribution yourself, or use our convenience binary package you can find here: * Download: http://wicket.apache.org/start/wicket-7.x.html#manually Upgrading from earlier versions ------------------------------- If you upgrade from 7.y.z this release is a drop in replacement. If you come from a version prior to 7.0.0, please read our Wicket 7 migration guide found at * http://s.apache.org/wicket7migrate Have fun! — The Wicket team ======================================================================== The signatures for the source release artefacts: Signature for apache-wicket-7.16.0.zip:
