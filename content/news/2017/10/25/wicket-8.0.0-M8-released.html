25 Oct 2017

The Apache Wicket PMC is proud to announce Apache Wicket 8.0.0-M8!

Apache Wicket is an open source Java component oriented web application framework that powers thousands of web applications and web sites for governments, stores, universities, cities, banks, email providers, and more. You can find more about Apache Wicket at https://wicket.apache.org

This release marks another minor release of Wicket 8. We use semantic versioning for the development of Wicket, and as such no API breaks are present breaks are present in this release compared to 8.0.0.

New and noteworthy

  • Issues WICKET-6457 and WICKET-6465 which caused the page store not to be cleared when session ends are now fixed.
  • This release has dropped Joda Time dependency (WICKET-6105). As consequence module wicket-datetime has been removed from Wicket distribution. Users are invited to migrate to new components based on Java 8 Date and Time APIs which have been added to module wicket-extensions. For those who needs to use old wicket-datetime classes we have migrated the code to WicketStuff project. They can be used adding the following dependency:
  • The experimental module http2-wicket now supports the final servlet 4.0 API specification.

Using this release

With Apache Maven update your dependency to (and don’t forget to update any other dependencies on Wicket projects to the same version):

Or download and build the distribution yourself, or use our convenience binary package

  • Source: 8.0.0-M8 source download
  • Binary: 8.0.0-M8 binary download

Upgrading from earlier versions

If you upgrade from 8.y.z this release is a drop in replacement. If you come from a version prior to 8.0.0, please read our Wicket 8 migration guide found at

  • http://s.apache.org/wicket8migrate

Have fun!

— The Wicket team
